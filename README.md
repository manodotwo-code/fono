# FonoVida - Landing Page Premium para Clínica de Fonoaudiologia

Este projeto é uma landing page de alta performance, moderna e responsiva para uma clínica de fonoaudiologia, desenvolvida com foco em estética premium e conversão.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e SEO otimizado.
- **Tailwind CSS** - Framework CSS para um design responsivo, moderno e de carregamento rápido.
- **Google Fonts** - Utilização das fontes *Plus Jakarta Sans* e *Playfair Display* para um visual sofisticado.
- **Phosphor Icons** - Conjunto de ícones minimalistas e modernos.
- **Animações Nativas** - Uso de keyframes e transições para uma experiência de usuário fluida.

## ✨ Diferenciais do Design (Tendências 2026)

- **Bento Grids:** Organização de conteúdo em blocos harmônicos.
- **Glassmorphism:** Elementos com aparência de vidro e desfoque de fundo.
- **Ambient Lighting:** Orbes de luz suaves e gradientes mesh no fundo.
- **Micro-interações:** Hover effects e animações de scroll que dão vida à página.
- **Foco em Conversão:** Botões de CTA (Call to Action) estratégicos e integração direta com WhatsApp.
